CSS Display

The CSS display property is used to define the rendering behavior of an element in terms of its box type and layout. It determines how an element is displayed on the web page. The display property can take various values, each affecting the element's layout and interactions differently.

1. display: block

block: The element generates a block-level box, which means it starts on a new line and stretches the full width of its containing element. Example: div, p, h1,

Div Box 0 ;display: block;
Above div box width 50% only, but its take the entire line

2. Display inline

Div Box - 1 ; display: inline;
Display inline : The element generates an inline-level box, which means it does not start on a new line and only takes up as much width as necessary.(height & width not affect). tags default inline display Examples are: span, a, strong

3. Display inline-block

Div Box - 2 ; display: inline-block;
The element generates an inline-level box, but it behaves as a block-level element regarding its layout. It doesn't start on a new line and can have width and height properties.

4. Display none

Div Box - 3; display: none;
The element is not displayed at all, making it effectively invisible and taking up no space in the layout. This is often used for hiding and showing elements dynamically using JavaScript.

5. Display flex

Div Box - 4; display: flex;
The element becomes a flex container, and its children become flex items. This allows for flexible layouts with the ability to easily control the arrangement of items along the main and cross axes.

6. Display grid

Div Box - 5; display: grid;
The element becomes a grid container, and its children become grid items. This allows for a two-dimensional layout system, providing both rows and columns.
